1. 프로젝트 개요
 WebJava, OracleDB(sql)을 이용. 기본 CRUD(Create Read Update Delete)가 작동하는 홈페이지 구현 
 - 칼로리를 부탁해 쇼핑페이지 참조
2. 프로젝트 환경
java 1.8 (Jdk. 1.8)
Oracle 11g
apache-tomcat-8.5.78
eclipse 2021-06

3 . 프로젝트 사용법
프로젝트에 ojdbc6.jar , Servers에 해당 프로젝트 db연결
sql문을 사용하여 db에 테이블 생성
index.jsp 부터 실행하여 회원가입 진행.
로그인하여 각 게시판 테스트
4 . 프로젝트 기능
 1.  회원가입 - 로그인
 2.  회원가입 정보 수정
 3.  로그인버튼 <-> 로그아웃버튼 구현
 4.  각 게시판 작성, 확인, 수정, 삭제 - 비밀번호 체크
5 . 프로젝트 결과물
회원가입
로그인 (로그인 실패 / 성공 ) / 로그아웃
게시판 게시글 작성, 확인 ( 성공 / 실패 )
게시판 게시글 수정, 삭제 ( 성공 / 실패 )
